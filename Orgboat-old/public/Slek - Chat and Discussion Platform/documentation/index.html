Slek

Chat and Discussion Platform

1 . Get Started

Slek allows people to communicate with each other in a digital environment. Allows you to send or receive digital documents such as text, video, photos, and files. It has many features with its modular structure.

In this document you will learn how to install and use Slek.

2 . File Structure

Initially, you should have a structure like this. 

- dist
  - css
  - icons
  - js
  - media
    - img
    - svg
- src (uncompiled files)
  - js
  - scss
- vendor (plugins)
- index.html

 

3 . Demos

You will have 3 different demos. Select the desired folder from the downloaded folder.

- default
- dark
- rtl

 

4 . Application structure

4.1 Navigation

Basic structure of navigation. 

<body>

<nav class="navigation">
    <div class="nav-group">
        <ul>
            <li>
                <a data-navigation-target="chats" class="active" href="#">
                    <i class="ti-comment-alt"></i>
                </a>
            </li>
            <li>
                <a data-navigation-target="friends" href="#" class="notifiy_badge">
                    <i class="ti-user"></i>
                </a>
            </li>
        </ul>
    </div>
</nav>

</body>

 

4.1 Logo placement in navigation

You can use your logo in any format.

<nav class="navigation">
    <div class="nav-group">
        <ul>
            <li>
                <a class="logo" href="#">
                    <!-- svg, img, png or jpeg -->
                </a>
            </li>
            
            ...

        </ul>
    </div>
</nav>

 

4.2 Separate links

Select some code if you want some links to appear at the beginning of some links. After the .brackets class, all links are pinned down.


<nav class="navigation">
    <div class="nav-group">
        <ul>
            ...

            <li class="brackets">
                <a href="#">
                    <i class="ti-star"></i>
                </a>
            </li>
            
            ...
        </ul>
    </div>
</nav>

 

4.3 Referrals

You can use the following structure to open other content (sidebars) by clicking on the links.

<nav class="navigation">
    <div class="nav-group">
        <ul>
            ...

            <li>
                <a data-navigation-target="chats" href="#">
                    <i data-feather="star"></i>
                </a>
            </li>
            
            ...
        </ul>
    </div>
</nav>


<!-- data-navigation-target="chats"  === #chats -->


<div class="content">

     <div class="sidebar-group">

          <div id="chats" class="sidebar">
                
          </div>
     </div>

</div>
