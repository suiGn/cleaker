# Cleaker(data, algorithm) → {string}

Generates a Digital Identifier (DID) for an object using a specified hashing algorithm.
Parameters:
Name Type Default Description
data Object The object to be hashed.
algorithm string SHA-256 The hashing algorithm to use ('SHA-256' or 'Keccak-256').
